# Google Clone

This project is a simple replica of the Google homepage, built using just HTML and CSS to showcase front-end development capabilities.

## Live Demo

[View the Google Clone](https://google-clone-tawny-ten.vercel.app/)

## Features

- Visually similar to the original Google homepage.
- Responsive design suitable for desktops, tablets, and mobiles.

## Technologies Used

- **HTML**: For structuring the webpage.
- **CSS**: For styling.

## Usage

Click on the "Live Demo" link above to view the project. Note: The search functionality is primarily for visual representation and might not provide actual search results.
